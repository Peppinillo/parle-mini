# Parle+ Mini (PWA Bundle)

Files included:
- `index.html` — the app (already wired for PWA)
- `manifest.webmanifest` — PWA manifest
- `sw.js` — service worker for installability/offline
- `icon-192.png`, `icon-512.png` — app icons

## Quick Deploy (GitHub Pages)
1. Create a new public repo on GitHub (e.g. `parle-mini`).
2. Upload all five files to the root of the repo.
3. In **Settings → Pages**, set **Deploy from branch**, **Branch: main**, **Folder: /(root)**.
4. Wait for the green check, then open: `https://<your-username>.github.io/parle-mini/`

## Install
- **Android/Chrome:** visit the URL → menu (⋮) → *Install app*.
- **iOS/Safari:** visit the URL → Share → *Add to Home Screen*.
